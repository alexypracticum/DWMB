# Changelog

## [0.4.0] — 2026-07-19

### Добавлено
- Редактор связей в UI (список исходящих/входящих, модальное окно добавления)
- Поиск сущности через AJAX при создании связи
- Удаление связей из UI
- Безопасность для GitHub: .env файлы, .gitignore, env_file в docker-compose
- Тесты: 42 unit-теста (layout, i18n, ontology, relationships, security)
- Конвертация онтологий в сущности (10 моделей, 30 шаблонов)
- Синхронизация CRUD с сущностями (model/template → entity)

### Исправлено
- Модель model_code → entity_code синхронизация при переименовании
- UUID() → uuid.uuid4() в add-projection
