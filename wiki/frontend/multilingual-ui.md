# Мультиязычный интерфейс

Система переводов UI в [[architecture/overview|DWMB]]: два уровня переводов, поддержка 7 языков.

## Архитектура переводов

### Уровень 1: UI-строки

**Хранение:** `app/services/i18n.py`

```python
TRANSLATIONS = {
    "ru": {
        "nav.home": "Главная",
        "nav.search": "Поиск",
        "btn.create": "Создать",
        "btn.save": "Сохранить",
        ...
    },
    "en": {
        "nav.home": "Home",
        "nav.search": "Search",
        "btn.create": "Create",
        "btn.save": "Save",
        ...
    }
}
```

**Подстановка:** Middleware `ThemeMiddleware`

```python
request.state.t = TRANSLATIONS.get(language_code, TRANSLATIONS["en"])
```

**Использование в шаблонах:**

```html
<button>{{ t.btn.save }}</button>
```

### Уровень 2: Данные сущностей

**Хранение:** таблица `entity_label`

```sql
SELECT label, description
FROM entity_label
WHERE entity_id = :entity_id
AND language_code = :language_code;
```

**Использование:**

- Метки сущностей на разных языках
- Переводы полей
- Локализованные описания

## Интерфейс управления переводами

### Статус

- Интерфейс позволяет добавлять только RU и EN поля
- Остальные языки (de, fr и др.) доступны в entity_label, но не в UI
- Нет массового управления переводами

### Требования из Задачи.txt

1. Единый интерфейс для управления переводами
2. Группировка полей по страницам
3. Массовое редактирование
4. Экспорт/импорт переводов

## Проблемы

### 1. Разделение интерфейса

Два параллельных подхода:
- i18n.py для UI-строк
- entity_label для данных сущностей

### 2. Ограничение языков в UI

Интерфейс позволяет работать только с ru/en. Остальные языки недоступны.

### 3. Нет CRUD для переводов

Нет удобного интерфейса для управления переводами.

### 4. Мультиязычность в CMS

Страницы CMS не поддерживают мультиязычность по умолчанию.

## Планы

- Единый интерфейс для управления переводами
- Расширение поддержки языков в UI
- Мультиязычность для CMS-страниц
- Автоматические переводы через AI (см. [[plugins/ai-plugin]])
- Экспорт/импорт переводов
