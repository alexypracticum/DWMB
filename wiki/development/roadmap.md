# Дорожная карта

Планы развития [[architecture/overview|DWMB]]: текущий статус и ближайшие приоритеты.

## Текущая версия

**v0.9.0** (июль 2026)

- Базовая архитектура работает
- 28+ таблиц в БД
- 7 плагинов
- 7 языков
- 160+ типов сущностей

## Приоритеты

### Приоритет 1: Философия "Всё как сущность"

Приведение системы в соответствие с философией проекта.

| Задача | Статус | Описание |
|--------|--------|----------|
| EntityKind CRUD | Частично | Создание/удаление через UI |
| ontology_model CRUD | Не начато | Полный CRUD для моделей |
| field_registry как entity | Не начато | Поля как сущности |
| media_asset как entity | Не начато | Файлы как сущности |
| page_registry как entity | Не начато | Страницы как сущности |
| Плагины как сущности | Не начато | Управление через UI |

### Приоритет 2: Мультиязычность

Расширение поддержки языков.

| Задача | Статус | Описание |
|--------|--------|----------|
| CRUD для переводов | Не начато | Интерфейс управления |
| 7 языков в UI | Частично | Только ru/en |
| Мультиязычность CMS | Не начато | Страницы с переводами |
| AI-переводы | Реализовано | Автоматические переводы |

### Приоритет 3: REST API

Полноценный REST API.

| Задача | Статус | Описание |
|--------|--------|----------|
| Префикс /api/v1/ | Не начато | Разделение API и UI |
| RESTful CRUD | Частично | Не для всех сущностей |
| OpenAPI документация | Не начато | Swagger |
| Аутентификация API | Не начато | JWT |

### Приоритет 4: UI/UX

Улучшение интерфейса.

| Задача | Статус | Описание |
|--------|--------|----------|
| Объединение страниц AI | Не начато | Одна страница |
| Центрирование сущности | Не начато | Ширина как у остальных |
| Графовая визуализация | Не начато | D3.js / Cytoscape.js |
| Автодополнение | Не начато | Поиск |

### Приоритет 5: Интеграции

Внешние интеграции.

| Задача | Статус | Описание |
|--------|--------|----------|
| Wikipedia API | Не начато | Импорт данных |
| IMDB API | Не начато | Фильмы |
| MusicBrainz API | Не начато | Музыка |
