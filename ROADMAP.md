### Цель: Безопасность TMDB, i18n, UI улучшения

**Выполнено:**
- [x] Исправление SQL-инъекции в TMDB импорте (ORM вместо raw SQL)

- [x] TMDB API интеграция (фильмы, люди, жанры, кредиты)

## v0.5.0 — Следующий этап (планирование)

- [ ] Интеграция IMDB через OMDb API

**Неделя 7-8: Тестирование и документация**
- [ ] Интеграционные тесты API (в контейнере)
