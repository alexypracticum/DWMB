## v0.5.0 — Следующий этап

### Цель: IMDB/TMDB импорт + Rate limiting + Визуализация

**Неделя 1: TMDB API (приоритет)**
- [ ] Интеграция TMDB API (фильмы, сериалы, люди)
- [ ] Импорт фильмов: название, год, жанр, режиссёр, актёры, постер, сборы
- [ ] Импорт людей: имя, фото, фильмография
- [ ] Автосоздание связей: directed_by, acted_in, produced_by
- [ ] UI: кнопка "Импорт из TMDB" на странице создания/редактирования

**Неделя 2: IMDB API (приоритет)**
- [ ] Интеграция IMDB через OMDb API (бесплатный ключ)
- [ ] Импорт: рейтинг, актёры, описание, постер
- [ ] Сопоставление с существующими сущностями
- [ ] Дедупликация при повторном импорте

**Неделя 3: Wikipedia + MusicBrainz**
- [ ] Wikipedia API (автозаполнение полей по URL)
- [ ] MusicBrainz API (исполнители, альбомы, треки)
- [ ] Кэширование результатов

**Неделя 4-5: Rate limiting**
- [ ] Middleware для ограничения по IP
- [ ] Лимиты: API 100/мин, страницы 300/мин
- [ ] Защита от brute-force (логин: 5 попыток/5 мин)
- [ ] Логирование превышений

**Неделя 6-7: Визуализация связей**
- [ ] D3.js или Cytoscape.js
- [ ] Граф связей на странице сущности
- [ ] Фильтрация по типам связей
- [ ] Интерактивное исследование

**Неделя 8: Тестирование и документация**
- [ ] Интеграционные тесты API
- [ ] Swagger/OpenAPI документация
- [ ] README с примерами использования
